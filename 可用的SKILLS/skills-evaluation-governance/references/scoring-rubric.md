# Scoring Rubric

## Score Bands

- 90-100: installable, well-triggered, eval-covered, operationally tracked
- 80-89: strong knowledge and workflow, needs eval or CI hardening
- 70-79: useful notes/templates, not yet a reliable skill
- 50-69: partial workflow, unclear trigger or output
- below 50: mostly raw notes

## Required Evidence

- required files exist
- frontmatter has `name` and `description`
- description names both task and trigger contexts
- body is concise and procedural
- references are linked from `SKILL.md`
- evals include positive and negative cases
- validation result is reported honestly

## Separate Scores

Always separate:

- notes/library score
- installable skill score
- Skill Bench readiness score
