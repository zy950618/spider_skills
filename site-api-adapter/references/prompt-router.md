# Prompt Router

## Output Shape

```json
{
  "task_type": "http|js_runtime|browser_diagnostic|waf|adapter|unsupported",
  "confidence": 0.0,
  "required_skill": "skill-name",
  "reason": "short reason",
  "inputs_needed": [],
  "verification": []
}
```

## Routing Rules

- Plain JSON API with static params: `http`
- Sign/token generated by JS: `js_runtime`
- Need page state or runtime variables: `browser_diagnostic`
- Incapsula/Reese84/WAF markers: `waf`
- Existing reverse notes need standardization: `adapter`
- Missing authorization or legally blocked: `unsupported`

Confidence below `0.7` should surface assumptions.
